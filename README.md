# Projeto INF1031

A ideia desse projeto é criar um protótipo de “Aquário inteligente” usando o a linguagem Lua e o microcontrolador ESP8266 D1 mini.
- Funcionalidades
  - Sensor de Temperatura
  - interface grafica
  - Exportação de dados em .csv
  - Controle de Bomba
